# 間詰め分断
制作者:HAIsan

## ダウンロード方法


## 概要
分断と間詰め、分断後の移動ができるエフェクトを追加するプラグイン 

## 使用方法
映像エフェクトの合成タブに「間詰め分断」が追加されます  
分断幅、分断角度、分断位置、分断後のズレ、分断後のズレの角度が設定可能です
※プレビュー上は変な挙動をしていてもエンコードの際は治っていることがほとんどです
※exo出力には対応していません
